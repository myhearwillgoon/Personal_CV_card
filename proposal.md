# 程权个人名片网页 - 项目提案 (Proposal)

## 项目概述

### 项目背景
程权先生需要一份专业的在线个人名片网页，用于展示个人职业信息、专业背景以及AI作品集视频。该网页将作为简历的重要组成部分，便于潜在雇主或合作伙伴通过互联网快速了解其专业能力和项目经验。

### 项目目标
创建一个现代化、响应式的个人名片网页，包含：
- 专业的个人信息展示
- 作品集视频集成
- 支持互联网访问（GitHub Pages 部署）
- 可直接嵌入简历或作为独立链接分享

---

## 需求摘要

### 核心需求
| 类别 | 需求描述 | 优先级 |
|------|----------|--------|
| 布局设计 | 三栏布局：左侧头像+信息、中间简介、右侧视频 | 高 |
| 个人信息 | 姓名、职位、邮箱等简历关键信息 | 高 |
| 作品集 | 集成3个AI项目演示视频 | 高 |
| 响应式 | 适配桌面、平板、手机多设备 | 高 |
| 部署 | GitHub Pages 公网访问 | 高 |

### 技术选型
- **前端技术**: HTML5 + CSS3 + JavaScript (原生)
- **部署平台**: GitHub Pages
- **响应式**: CSS Flexbox/Grid + Media Queries
- **视频托管**: GitHub 仓库（MP4格式）

---

## 设计理念

### 视觉风格
- **风格定位**: 简约、专业、科技感
- **色彩方案**: 深色主题（专业感）或浅色主题（清新感），建议主色调为蓝色系（科技）
- **字体选择**: 中文字体使用思源黑体/微软雅黑，英文使用 Inter/Roboto

## 项目范围

### 包含范围 (In Scope)

## 风险评估

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| GitHub Pages 访问问题 | 高 | 使用 CDN 加速，备选 Netlify |
| 视频文件过大 | 中 | 压缩视频，或使用视频托管平台 |
| 移动端适配问题 | 中 | 多设备测试，渐进增强 |
